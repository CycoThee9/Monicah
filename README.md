# 🎂 MONICAH KANINI - Birthday Celebration Website 🎂

A beautiful, poetic GitHub Pages website celebrating the birthday and personality of Monicah Kanini.

## 🌟 Features

This website is a creative and romantic tribute featuring:

### 📖 Pages

1. **Home Page (`index.html`)** - Main celebration page featuring:
   - Poetic greeting and birthday wishes
   - Personality highlights with beautiful quality cards
   - "Your Essence" section celebrating her characteristics
   - Birthday wishes and messages
   - Quick navigation to poetry collections

2. **Whispers of the Heart (`poetry1.html`)** - First poetry collection including:
   - "In Your Eyes, I Found Home"
   - "The Sweetness of Your Name"
   - "Stolen Moments"
   - "You Are the Reason"

3. **Love's Melody (`poetry2.html`)** - Second poetry collection featuring:
   - "A Heart Intertwined"
   - "The Alchemy of Your Touch"
   - "Endless Like the Ocean"
   - "Perfect in Imperfection"
   - "Forever in a Glance"

## 🎨 Design Elements

- **Color Scheme**: Elegant gold (#d4a574), rose (#f5a3c7), and deep navy background

## 🚀 Deployment on GitHub Pages

This website is ready to be deployed on GitHub Pages:

1. Go to your repository settings
2. Navigate to "Pages" section
3. Select the branch to deploy from (usually `main` or `master`)
4. Wait for the deployment to complete
5. Your website will be available at: `https://CycoThee9.github.io/Monicah/`

## 🎭 Personalization

To customize this website:

1. **Change Colors**: Modify the CSS variables in `styles.css` (`:root` section)
2. **Update Text**: Edit the poetic content in HTML files
3. **Add Photos**: Create an `images` folder and reference images in HTML
4. **Customize Poems**: Replace poems with your own verses
